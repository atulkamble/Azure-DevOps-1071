## Manage Containers

### Stop Container

```bash
sudo docker container stop <container-id>
```

Example:

```bash
sudo docker container stop 6e1f7fcd9491
```

### Start Container

```bash
sudo docker container start 6e1f7fcd9491
```

### Remove Container

```bash
sudo docker container rm <container-id>
```

### Remove Image

```bash
sudo docker rmi <image-id>
```

## Access Jenkins Container


### Get Initial Admin Password

