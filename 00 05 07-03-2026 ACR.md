# Azure Container Registry (ACR) – Python Docker Project

## Project Repository

[https://github.com/atulkamble/Python-ACR-Project.git](https://github.com/atulkamble/Python-ACR-Project.git)

This project demonstrates how to:

* Build a **Python Flask application**
* Create a **Docker image**
* Push the image to **Azure Container Registry (ACR)**
* Use the container image for deployments

---

# 1. Fork the Repository

Fork the repository to your GitHub account.

```
https://github.com/atulkamble/Python-ACR-Project.git
```

Click **Fork** on GitHub.

---

# 2. Clone the Repository

```bash
git clone https://github.com/atulkamble/Python-ACR-Project.git
cd Python-ACR-Project
code .
```

---

# 3. Verify Python Environment

Check Python and Pip versions.

```bash
python --version
pip --version
```

---

# 4. Install Application Dependencies

Install required Python libraries.

```bash
pip install -r requirements.txt
```

---

# 5. Run the Python Application

```bash
python app.py
```

Open browser:

```
http://localhost:5000
```

The Flask application should now be running locally.

---

# 6. Login to Azure

```bash
az login
```

This opens the browser for Azure authentication.

---

# 7. Login to Azure Container Registry (ACR)

```bash
az acr login --name atulkamble
```

This connects Docker to Azure Container Registry.

---

# 8. Build Docker Image

Build the container image for **ARM64 architecture**.

```bash
sudo docker buildx build \
--platform linux/arm64/v8 \
-t atulkamble.azurecr.io/ethans/pythonapp:latest \
--load .
```

Explanation:

| Option   | Meaning                 |
| -------- | ----------------------- |
| buildx   | Advanced Docker builder |
| platform | Target CPU architecture |
| -t       | Image tag               |
| --load   | Load image locally      |

---

# 9. Verify Docker Image

```bash
sudo docker images
```

Example output

```
atulkamble.azurecr.io/ethans/pythonapp   latest
```

---

# 10. Login to ACR via Docker

```bash
sudo docker login atulkamble.azurecr.io
```

Enter:

```
Username: atulkamble
Password: <ACR token or password>
```

---

# 11. Push Image to Azure Container Registry

```bash
sudo docker push atulkamble.azurecr.io/ethans/pythonapp:latest
```

This uploads the Docker image to **Azure Container Registry**.

---

# 12. Verify Image in Azure

Go to:

```
Azure Portal
→ Container Registry
→ atulkamble
→ Repositories
→ ethans/pythonapp
```

You will see the pushed image **latest tag**.

---

# Architecture

```
Developer Machine
      │
      │ Build Docker Image
      ▼
Docker Image
      │
      │ Push
      ▼
Azure Container Registry (ACR)

      │ Pull
      ▼
AKS / Azure App Service / Container Instances
```
