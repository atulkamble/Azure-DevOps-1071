# 1. What is Kubernetes

**Kubernetes (K8s)** is an **open-source container orchestration platform** used to **automate deployment, scaling, and management of containerized applications**.

It was originally developed by **Google** and is now maintained by the **Cloud Native Computing Foundation (CNCF)**.

### Key Characteristics

* Production-grade container orchestration
* Automated deployment
* Auto scaling
* Self healing
* High availability
* Load balancing
* Rolling updates

# 6. Kubernetes Core Concepts

## Pod

A **Pod** is the **smallest deployment unit in Kubernetes**.

* Runs one or more containers
* Containers share network and storage

Example

```
Pod
 └── Container
```

---

## Node

A **Node** is a **physical machine or virtual machine** that runs pods.

Nodes contain:

* kubelet
* container runtime
* kube-proxy

Example

```
Node
 ├── Pod
 ├── Pod
 └── Pod
```

---

## Cluster

A **Cluster** is a **group of nodes** managed by Kubernetes.

```
Cluster
 ├── Node 1
 │    ├── Pod
 │    └── Pod
 │
 ├── Node 2
 │    └── Pod
 │
 └── Node 3
      └── Pod
```

---

# 7. Kubernetes Deployment Flow

The application flows through the following architecture:

```
Code
   ↓
Container (Docker)
   ↓
Pod
   ↓
Node
   ↓
Cluster
```

Example

```
Application Code
       ↓
Docker Container
       ↓
Kubernetes Pod
       ↓
Node (VM)
       ↓
Cluster
```

# 8. Key Kubernetes Features

## Self Healing

Kubernetes automatically fixes issues.

Example:

* Restart containers
* Replace failed pods
* Reschedule pods to healthy nodes

---

## High Availability (HA)

Kubernetes distributes workloads across nodes.

Benefits:

* Fault tolerance
* No downtime
* Reliable production systems

# 10. Install Minikube (Windows)

Open **PowerShell as Administrator**

---

### Install WSL

```bash
wsl --install
```

---

### Install Chocolatey

Install from

```
https://chocolatey.org/install
```

---

### Install Minikube

```bash
choco install minikube
```

---

### Start Kubernetes Cluster

```bash
minikube start
```

This will create a **single-node Kubernetes cluster** locally.
